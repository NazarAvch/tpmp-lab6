# Задание 5 — Математика для детей на Swift

## Описание (Вариант 1)
Приложение для изучения умножения и деления однозначных и двузначных чисел. Тип числа и тип операции задаётся. Значения чисел вводятся пользователем.

## Сцены
1. **SettingsViewController** — выбор типа числа и операции, кнопка «Начать»
2. **GameViewController** — вопрос, поле для ответа, проверка, результат

## Особенности
- AutoLayout и StackView для адаптации под разные экраны
- Локализация на 3 языка
