# Задание 1 — Управление фоном на Swift

## Описание
iOS-приложение с объектом Switch для изменения фона контроллера. В положении «Вкл» — изображение bg1.jpg, в положении «Выкл» — bg2.jpg.

## Элементы интерфейса
- UISwitch — переключатель
- UILabel — индикатор текущего фона

## Функциональность
- При запуске: Switch OFF, фон bg2
- Переключение: меняет фон и текст индикатора
