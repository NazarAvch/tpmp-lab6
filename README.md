# Задание 2 — Погода на Objective-C

## Описание
Приложение показа погоды для городов. При выборе города выводит изображение герба, страну и температуру. Цвет температуры зависит от погоды.

## Данные (Вариант 1)
- Литва: Вильнюс (-5°C), Каунас (-3°C)
- Польша: Варшава (2°C), Краков (4°C)

## Элементы интерфейса
- UIPickerView — выбор города
- UILabel — название города, страна, температура
- UIImageView — герб города
- UIButton — обновить
