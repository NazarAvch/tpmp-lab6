# Задание 3 — Калькулятор калорий и ИМТ на Swift

## Описание
Приложение для расчёта индекса массы тела (ИМТ) и дневной нормы калорий по формуле Харриса-Бенедикта.

## Элементы интерфейса
- UITextField — возраст, рост (см), вес (кг)
- UISegmentedControl — пол (мужской/женский), активность (низкая/средняя/высокая)
- UIButton — рассчитать
- UILabel — результат (ИМТ, BMR, дневная норма)

## Формулы
- ИМТ = вес / (рост в метрах)²
- BMR (Харрис-Бенедикт)
